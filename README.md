# Romantic Composer

This is a small Next.js (App Router) project with Tailwind CSS that provides a simple "Romantic Composer" — upload images, add text and icons, drag them around, and export a PNG.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push this repository to GitHub and deploy to Vercel (https://vercel.com/new) — Vercel will detect Next.js and deploy automatically.
